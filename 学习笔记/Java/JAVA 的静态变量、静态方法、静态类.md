#### 1. Java静态对象和非静态对象区别

 拥有静态变量和静态方法的对象都属于静态对象，它与非静态对象的差别需要做个说明。

* 属性
  * 静态对象：共有
  * 非静态对象：各对象独立拥有
* 内存分配
  * 静态对象：内存空间上固定
  * 非静态对象：空间在各个附属类里面分配

静态对象好处：

* 静态对象数据在全局上是唯一的，一改都改。
* 引用方便
* 保持数据唯一性
* static final用来修饰成员变量和成员方法，可简单理解为“全局常量”。对于变量，表示一旦给值就不可修改；对于方法，表示不可覆盖。
